# 🎬 剧集营销方案生成器

专业的剧集营销方案生成工具

## 🚀 快速部署到Vercel

### 方法1：通过GitHub（推荐）

1. 注册GitHub账号：https://github.com
2. 创建新仓库（Repository）
3. 上传这个文件夹的所有文件
4. 访问 https://vercel.com 用GitHub登录
5. 导入项目，自动部署
6. 获得免费网址！

### 方法2：拖拽部署

1. 访问 https://vercel.com
2. 注册/登录
3. 点击 "Add New Project"
4. 选择 "Deploy from Git" 或直接上传文件夹
5. 完成！

## 📦 文件说明

- `index.html` - 主程序（唯一需要的文件）
- `README.md` - 本说明文档

## 💡 使用说明

部署后，任何人都可以通过网址访问使用！

完全免费，无需数据库，无需后端服务！
