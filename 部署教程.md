# Cloudflare Worker 部署教程

## 方式1：通过网页界面（推荐，最简单）

### 步骤1：登录Cloudflare
访问：https://dash.cloudflare.com/
用您的邮箱登录（部署Pages时用的邮箱）

### 步骤2：创建Worker
1. 左侧菜单 → Workers & Pages
2. 点击 "Create application"
3. 选择 "Create Worker"
4. 名字：drama-marketing-api
5. 点击 "Deploy"

### 步骤3：编辑代码
1. 点击 "Edit code"
2. 删除所有默认代码
3. 打开本地文件：worker.js
4. 复制所有内容，粘贴到编辑器
5. 点击 "Save and deploy"

### 步骤4：设置API Key
1. 回到Worker主页
2. 点击 "Settings" 标签
3. 找到 "Variables" 部分
4. 点击 "Add variable"
   - Name: DASHSCOPE_API_KEY
   - Value: sk-sp-f65d5e54bb3049e6a92b7a4de6617ef7
   - 勾选 "Encrypt"
5. 点击 "Save"

### 步骤5：获取Worker URL
- Worker页面会显示URL
- 格式：https://drama-marketing-api.xxx.workers.dev
- 复制这个URL

## 方式2：通过命令行

### 前置条件
```bash
# 安装Node.js（如果没有）
# 下载：https://nodejs.org/

# 安装wrangler
npm install -g wrangler
```

### 部署步骤
```bash
# 1. 进入项目目录
cd ~/Documents/GitHub/drama-marketing-generator

# 2. 登录Cloudflare
wrangler login
# 会打开浏览器，点击授权

# 3. 部署Worker
wrangler deploy

# 4. 设置环境变量
wrangler secret put DASHSCOPE_API_KEY
# 粘贴：sk-sp-f65d5e54bb3049e6a92b7a4de6617ef7
# 按回车

# 5. 查看部署结果
wrangler deployments list
```
